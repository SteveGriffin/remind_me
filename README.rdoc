== Remio

This is the repository for the remio service, hosted at remio.me

The service is designed to send reminder messages using Twilio services, and currently has the following capabilities:

* SMS messaging

* Voice Messaging - Receive a voice call that reads your message back to you!

* Mass Messaging - Send a mass reminder to a group of friends, co-workers, etc.

* Easy Google OAuth login, no personal information or passwords required
